# Vue 3 with TypeScript (WIP)
A demo project outlining how to use Vue 3 with TypeScript to build powerful apps.

## What is used?
- Vue 3
- TypeScript 3.9
  - with type imports
- Babel 7
- Webpack 5
- ESLint
- Hot Module Replacement
